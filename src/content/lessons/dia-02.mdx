## ü§ñ Los 5 Tipos de Datos Esenciales

En Java hay muchos tipos, pero hoy nos enfocaremos en los 5 que usar√°s el 90% del tiempo. Vamos a usar un **Robot** como ejemplo:

### 1. `int` (Enteros)
Para n√∫meros **sin decimales**.
- **Ejemplos**: Edad, cantidad de vidas, n√∫mero de ruedas.
- **Robot**: `int ruedas = 4;`

### 2. `double` (Decimales)
Para n√∫meros **con decimales** (mayor precisi√≥n).
- **Ejemplos**: Precio, altura, porcentaje de bater√≠a.
- **Robot**: `double bateria = 98.5;`

### 3. `String` (Texto)
Para **cadenas de texto**. Siempre va entre **comillas dobles** `""`.
- **Ejemplos**: Nombre, direcci√≥n, mensaje.
- **Robot**: `String nombre = "RoboCop";`

### 4. `boolean` (L√≥gico)
Solo puede valer `true` (verdadero) o `false` (falso). Es como un interruptor.
- **Ejemplos**: ¬øEst√° encendido?, ¬øEs admin?, ¬øTiene acceso?
- **Robot**: `boolean encendido = true;`

### 5. `char` (Car√°cter)
Para un **solo s√≠mbolo**. Siempre va entre **comillas simples** `''`.
- **Ejemplos**: Inicial, calificaci√≥n (A, B, C), g√©nero (M/F).
- **Robot**: `char modelo = 'X';`

## üìè Reglas para Nombres (Sintaxis)

Java es estricto. Sigue estas reglas para nombrar tus variables:

1. **Empieza con min√∫scula**: `nombre`, `edad`.
2. **Usa camelCase**: Si son dos palabras, la segunda empieza con may√∫scula.
   - ‚úÖ `nombreCompleto`
   - ‚úÖ `velocidadMaxima`
   - ‚úÖ `numeroDeVidas`
   - ‚ùå `Nombre_Completo` (Estilo Python/C)
   - ‚ùå `nombrecompleto` (Dif√≠cil de leer)
3. **S√© descriptivo**:
   - ‚úÖ `int puntosDeVida = 100;`
   - ‚ùå `int x = 100;` (¬øQu√© es x?)

## üö® Errores Comunes

### 1. `Incompatible types`
```java

```
**Soluci√≥n**: Aseg√∫rate de que el valor coincida con el tipo de la caja.

### 2. Olvidar las comillas en String
```java

```
**Soluci√≥n**: Texto siempre va entre comillas dobles: `String nombre = "Oscar";`

### 3. Usar comillas simples para String
```java
String nombre = 'Oscar'; // Error
```
**Soluci√≥n**: Comillas simples `''` son solo para `char` (un car√°cter). Strings usan `""`.

### 4. Declarar la misma variable dos veces
```java

int x = 10; // Error: variable x is already defined
```
**Soluci√≥n**: Solo escribe el nombre para actualizar: `x = 10;`
