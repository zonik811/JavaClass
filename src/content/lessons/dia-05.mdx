## üö¶ 1. La Sentencia `if` (Si...)

El `if` es como un guardia de seguridad. Solo deja pasar el c√≥digo si se cumple una condici√≥n (es decir, si es `true`).

### Sintaxis
```java

    // C√≥digo que se ejecuta SI la condici√≥n es verdadera
}
```

### Ejemplo
```java

    System.out.println("¬°Bater√≠a baja! Conecta el cargador.");
}
```
*Si `bateria` fuera 50, el mensaje NUNCA se imprimir√≠a.*

## üõ§Ô∏è 2. `else` (Si no...) y `else if`

¬øQu√© pasa si la condici√≥n es falsa? Usamos `else` (si no) para definir un "Plan B".

### Estructura `if - else`
```java

    System.out.println("Eres mayor de edad.");

    System.out.println("Eres menor de edad.");
}
```

### Estructura `if - else if - else` (M√∫ltiples opciones)
Imagina un sem√°foro. No es solo "Verde" o "No Verde". Hay opciones intermedias.

```java

    System.out.println("Alto total.");
} else if (colorSemaforo.equals("AMARILLO")) {
    System.out.println("Precauci√≥n, baja velocidad.");

    System.out.println("Avanza.");

    System.out.println("Color desconocido, sem√°foro roto.");
}
```

## üö´ Errores Comunes

### 1. El punto y coma traicionero
```java
if (edad > 18); { // <--- ¬°ERROR! Ese ; termina el if inmediatamente.
    System.out.println("Esto se imprime SIEMPRE");
}
```
**Regla**: Los `if` no llevan `;` antes de la llave `{`.

### 2. Comparar Strings con `==`
Como vimos en el Quiz, para comparar texto se usa `.equals()`.
```java
String clave = "1234";
if (clave == "1234") // MAL (A veces funciona, a veces no)
if (clave.equals("1234")) // BIEN (Siempre funciona)
```

## üöÄ Mini-Proyecto: Calculadora B√°sica

Vamos a combinar **Scanner** (D√≠a 4) con **If/Else** (D√≠a 5) para crear una calculadora real.

**Requisitos**:
1. Pedir el **primer n√∫mero**.
2. Pedir el **segundo n√∫mero**.
3. Pedir la **operaci√≥n** (1: Suma, 2: Resta, 3: Multiplicaci√≥n, 4: Divisi√≥n).
4. Mostrar el resultado correcto seg√∫n la elecci√≥n.

<div slot="hint1">
  Usa `scanner` para pedir los datos. Luego usa una cadena de `if - else if` mirando la variable de la operaci√≥n.
</div>

<div slot="solution">
```java

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 1. Pedir datos
        System.out.println("Ingresa el primer n√∫mero:");
        double num1 = scanner.nextDouble();

        System.out.println("Ingresa el segundo n√∫mero:");
        double num2 = scanner.nextDouble();

        System.out.println("Elige operaci√≥n: 1-Suma, 2-Resta, 3-Multi, 4-Div");
        int op = scanner.nextInt();
        
        double resultado = 0;

        // 2. L√≥gica de decisi√≥n
        if (op == 1) {
            resultado = num1 + num2;
            System.out.println("Suma: " + resultado);
        } else if (op == 2) {
            resultado = num1 - num2;
            System.out.println("Resta: " + resultado);
        } else if (op == 3) {
            resultado = num1 * num2;
            System.out.println("Multiplicaci√≥n: " + resultado);
        } else if (op == 4) {
            if (num2 != 0) {
                 resultado = num1 / num2;
                 System.out.println("Divisi√≥n: " + resultado);
            } else {
                 System.out.println("Error: No se puede dividir por cero.");
            }
        } else {
            System.out.println("Operaci√≥n no v√°lida.");
        }
        
        scanner.close();
    }
}
```
