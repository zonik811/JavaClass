## âš–ï¸ 2. Operadores de ComparaciÃ³n

Estos responden preguntas con **Verdadero (`true`)** o **Falso (`false`)**.
Son los "sensores" de tu programa.

| SÃ­mbolo | Significado | Ejemplo | Resultado |
| :---: | :--- | :--- | :--- |
| `>` | Mayor que | `10 > 5` | `true` |
| `<` | Menor que | `3 < 8` | `true` |
| `>=` | Mayor o igual | `10 >= 10` | `true` |
| `<=` | Menor o igual | `5 <= 2` | `false` |
| `==` | **Igual a** | `5 == 5` | `true` |
| `!=` | **Diferente de** | `5 != 5` | `false` |

### âš ï¸ Â¡Cuidado!
* `=` (uno solo) es para **ASIGNAR** valor (`x = 5`).
* `==` (dos) es para **COMPARAR** (`x == 5`).

## ğŸ§  3. Operadores LÃ³gicos

Estos sirven para combinar varias condiciones.
Ejemplo: "Si tengo dinero **Y** la tienda estÃ¡ abierta".

| Operador | Nombre | LÃ³gica | Ejemplo |
| :---: | :--- | :--- | :--- |
| `&&` | **AND** (Y) | Todo debe ser verdadero | `true && true` = `true` |
| `||` | **OR** (O) | Al menos uno debe ser verdadero | `true || false` = `true` |
| `!` | **NOT** (NO) | Invierte el valor | `!true` = `false` |

* `(5 > 3) && (2 < 4)` -> `true` Y `true` -> **`true`**
* `(5 > 10) || (2 < 4)` -> `false` O `true` -> **`true`**
